# Kirti's Portfolio

A modern, responsive personal portfolio website built with Flutter.

## Features
- Responsive Design (Mobile, Tablet, Desktop)
- Dark Mode Theme
- Interactive UI Elements

## Getting Started

1.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```

2.  **Run Locally**:
    ```bash
    flutter run -d chrome
    ```

## Customization

### Personal Data
Update `lib/utils/constants.dart` with your personal information (Name, Role, Bio, Email).

### Sections
- **Experience**: Edit `lib/sections/experience_section.dart`.
- **Projects**: Edit `lib/sections/projects_section.dart`.
- **Skills**: Edit `lib/sections/about_section.dart`.

### Colors & Theme
Edit `lib/utils/constants.dart` to change the color palette.
